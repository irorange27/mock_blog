Hello, world!

你可以通过克隆 [本项目](https://github.com/irorange27/mock_blog) 并在 posts 文件夹下进行编辑来发布自己的博客。

## 快速开始

安装依赖

```bash
pnpm install
```

```bash
pnpm dev
```

## 部署

```bash
pnpm build
```

## 预览

```bash
pnpm preview
```
